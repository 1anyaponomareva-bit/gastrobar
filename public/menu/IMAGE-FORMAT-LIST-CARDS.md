# Формат картинок для горизонтальных карточек в списке (Бар / Меню)

Чтобы все карточки были одинаковыми по высоте и ширине, в коде задана **фиксированная высота карточки** и **фиксированная зона под картинку** справа. Картинка масштабируется с **object-fit: contain** — **без обрезки**, коктейль/блюдо целиком помещается в зону; по краям при необходимости остаётся чёрный фон.

## Размер зоны картинки на экране

- **Высота карточки:** 22dvh (минимум 100px, максимум 140px) — примерно **100–140 px** в зависимости от телефона.
- **Ширина зоны картинки:** 42% ширины экрана (минимум 90px, максимум 160px). На типичном телефоне (ширина ~360px) это **~150 px**.

То есть зона картинки на экране имеет пропорции примерно **1 : 1** (квадрат) или чуть выше (например 150×130 px → ~1.15 : 1).

## Рекомендуемый формат исходных картинок

Картинка не обрезается (contain), поэтому подойдут и квадрат, и вертикальный формат — главное, чтобы объект был по центру:

| Параметр        | Значение |
|-----------------|----------|
| **Соотношение сторон** | **1 : 1 (квадрат)** или **4 : 3** (горизонтально) |
| **Размер в пикселях**  | **400×400 px** (квадрат) или **480×360 px** (4:3). Для Retina можно **800×800** или **960×720** |
| **Формат файла**       | JPG или PNG |
| **Имя файла**          | Любое, путь указывается в `imageList` в `src/data/menu.ts` |

## Где должен быть стакан/коктейль на картинке

В коде используется **object-position: 50% 50%** — по одной линии выравнивается **центр кадра**. Поэтому:

- **По горизонтали:** стакан должен быть **строго по центру** изображения (50% от левого края).
- **По вертикали:** середина стакана — **по центру** по высоте кадра (50%).

Если стакан в кадре смещён (например, вправо), в приложении он будет «уезжать» относительно других карточек. Подробный промпт и ТЗ — в файле **IMAGE-SPEC-PROMPT.md**.

## Соотношение сторон

- В карточке картинка выводится с `object-fit: contain` и центрированием (`object-position: 50% 50%`) — **ничего не обрезается**, коктейль/блюдо целиком в кадре.
- Подойдут квадрат 1:1, вертикальный 3:4 (для высоких бокалов) и др.; стакан/объект должен быть по центру кадра для ровной линии в списке.

## Где что подставлять

- **В списке (горизонтальные карточки):** используется поле `imageList`, если оно задано, иначе `image`. Для Б-52 уже задано `imageList: "/menu/b52-horizontal-hero.png"`.
- **В полноэкранном просмотре:** всегда используется только `image` (основное фото).

Итого: делай картинки для списка **квадрат 400×400 px** (или 800×800 для чёткости), клади в `public/menu/` и прописывай путь в `imageList` в меню.
